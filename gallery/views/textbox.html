Textbox

A traditional style GDS text input, with a helper macro to create it. The id for the control uses a value designed to work with the error list component.

{{ trade.textbox("Simple", label="Description", hint="This is a hint", value="Hello Mum", id="text-1", class="custom-class") }}

HTML

<div class="form-group custom-class" id="Simple-wrapper">
  <label class="form-label-bold" for="text-1">Description
    <span class="form-hint">This is a hint</span>
  </label>
  <input id="text-1" class="form-control" name="Simple" type="text" value="Hello Mum">
</div>

Nunjucks Macros

{% raw %}{{ trade.textbox("Simple", label="Description", hint="This is a hint", value="Hello Mum", id="textarea-1", class="custom-class") }}{% endraw %}
Parameter Required Description
name Y the name for the field
label Y The label for the field
value Y The text value for the field
hint N Optional hint to show below the label
id N If you wish the field to be assigned an ID pass it here
class N Pass an extra class value to add to the markup, e.g. selected
error N If there is an error with the field, set the error string here.
