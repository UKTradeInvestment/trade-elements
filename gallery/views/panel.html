Sub panel

Panels are a way to group sections within a section, and their variant, panel-border, allows the sub sections to be visually indented. GDS is a little ambiguous how to handle errors

For trade the style is slightly modified to it clear, consistent and work with the macros to create field layouts. If a field in a section has an error the 'pane' should have an 'error' class. The fields within the panel will still have an error in it's form group but the left red border styling for this will not be shown, to avoid 2 borders.

As fields in the sub panel are related to a top field, they should all be contained in a fieldset.
