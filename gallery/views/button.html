Parameter Required Description
text Y The text to show on the button
id N If you wish the button to be assigned an ID pass it here
class N Pass an extra class value to add to the markup, e.g. selected
variation N In addition to the default green button styled by GDS, trade element add variations, supported values include: secondary, warning, disabled and link. Note that passing 'disabled' will also set the button so that it cannot be used.
type N Used to specify if a button is a 'button' or 'submit', defaults to 'button'
