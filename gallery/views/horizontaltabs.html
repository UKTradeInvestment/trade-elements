Horizontal Tabs

Horizontal tab panel to allow a header/detail type view

The horizontal tab is a self activating control. The Javascript for this control requires markup in the structure documented below. If no tab is selected then the first tab is auto selected. When the control loads the active tab is displayed and all over tab content is hidden. The URL is updated with a # pointing to the tab to be displayed so if the page is reloaded the correct tab is shown.

If Javascript is not enabled, all the tab content is visible and clicking the tab links move to that content.

<div class="tabs tabs--inline js-tabs">

    </li>

  </ul>
  <div id="tab1" class="tabs-panel">
    <div class="tabs-panel-inner">This is tab 1</div>
  </div>
  <div id="tab2" class="tabs-panel">
    <div class="tabs-panel-inner">This is tab 2</div>
  </div>
</div>
