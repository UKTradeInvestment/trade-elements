Dropdown

A standard select control styled for GDS style forms.

HTML

<div class="form-group " id="dropthing-wrapper">
    <label class="form-label-bold" for="dropthing">
      Example dropdown
      <span class="form-hint">Just a bit of fun</span>
    </label>
    <select id="dropthing" class="form-control" name="dropthing">
      <option value="">Pick a value</option>
      <option value="one" selected="selected">one</option>
      <option value="two">two</option>
      <option value="three">three</option>
    </select>
  </div>

Parameter Required Description
name Y The prefix name for the form field
label Y The text to display as a label for this field
hint N Optional text below label to give the user more help
emptyLabel N An optional option to show when there is no selected value
value N The current selected value
options Y An array of possible options to display and select. This is currently an array of strings. Later support will be provided for key value pairs.
id N An optional id value to use on a wrapper around the markup
class N A optional class name to add on the wrapper around the markup
error N An optional string describing an error to display for the field.
