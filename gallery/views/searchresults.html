Search Results

Search result has a collect of css classes to help display a search summary and a list of individual search results. There are no macros for this component as the layout of each search result may differ or need flexibility to show different markup for different entity types.

For each result the title should have a class 'result-title' and each distinct piece of data with that should be wrapped with a 'meta', classes like 'meta--ch' are outside the scope of this control but show how aditional styling could be added.
