Save bar

A common used pattern to provide a save and cancel button at the bottom of a form.

{{ trade.save(backUrl ='/') }}

HTML

<div class="save-bar">
  <button class="button button--save" type="submit">Save</button>
  <a class="button-link button--cancel js-button-cancel" href="/">Cancel</a>
</div>

Nunjucks Macros

{% raw %}{{ trade.save(backUrl ='/') }}{% endraw %}
Parameter Required Description
backUrl Y The url to use for the cancel button
buttonText N Alternative text to use instead of 'Save' on the save button, e.g. Submit
