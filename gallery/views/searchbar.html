Search bar

A Google like search bar. This control has two 'types', it can either be a regular searchbar displayed below, or a 'header' searchbar which is suited to use in the GDS style header, and is more compact.

{{ trade.searchbar("search", label="Search", value="fred") }}

HTML

<form class="searchbar" action="/search" method="get" role="search">
  <div class="searchbar__wrapper">
    <label class="searchbar__label" for="search-term">Search for company name or contact</label>
    <input class="searchbar__input form-control" id="search-term" type="search" name="term" value="" placeholder="Search for company name or contact">
    <input class="searchbar__submit" type="submit" value="Search">
  </div>
</form>

Nunjucks Macros

{% raw %}{{ trade.searchbar("search", label="Search", value="fred") }}{% endraw %}
Parameter Required Description
name Y The url to use for the cancel button
label Y A label to use to indicate what the search field is for, note that this is hidden from view and rendered solely for the purpose of accessibility
value N An existing value to populate the search field is there is one, e.g. the user has landed on a results page, show the current term
type N By default the search bar is styled for use in the main part of the page, optionally the developer can pass a value of 'header' to mutate the style to suit usage in a compact space such as the header.
action N The action attribute for the search form, this is a url. If no value is specified then '/search' is used.
submitLabel N A text label for the search button, this defaults to 'Search'
