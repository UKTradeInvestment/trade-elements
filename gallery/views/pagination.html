Pagination

Pagination allows movement through long lists of data of results. There is no macro to create the markup for this but by using the classes demonstrated it it possible to create the correctly styled layout.

2
3 Next

HTML

<div class="pagination">
  <div>
    <a href="/search?term=fred&page=1" class="pagination__page">Previous</a>
    <a href="/search?term=fred&page=1" class="pagination__page">1</a>
    <div class="pagination__page">2</div>
    <a href="/search?term=fred&page=3" class="pagination__page">3</a>
    <a href="/search?term=fred&page=3" class="pagination__page">Next</a>
  </div>
</div>
