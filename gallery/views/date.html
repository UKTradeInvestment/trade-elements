Date

Allow the user to enter a date in UK format, dd/mm/yyyy. The control accepts a string in dd/mm/yyyy format and split it into 3 fields for the user to edit.

{{ trade.date('dob', label="Date of birth", value="07/01/1971") }} {{ trade.date('startdate', label="Start date", value="2016-11-10T13:10:01.079Znpm ") }}

Nunjucks Macros

{% raw %}{{ trade.date('dob', label="Date of birth", value="07/01/1971") }}{% endraw %}
Parameter Required Description
name Y The prefix name for the form field
label Y The text to display as a label for this field
value N The date value in the format 'dd/mm/yyyy' or 'yyyy-mm-ddThh:MM:ss' or 'yyyy-mm-dd', leave empty if there is no existing value. You may also pass a native JS date
id N An optional id value to use on a wrapper around the markup
class N A optional class name to add on the wrapper around the markup
error N An optional string describing an error to display for the field.
